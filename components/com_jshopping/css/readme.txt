1) You need install node.js
2) Open a console
3) "npm install" in this directory

If you don't need load a minified files and with map, you have to go to : admin->Configuration->Other config->shop mode : Developer

If you need to build a files: 
    - npm run build
If you need watching for files and auto building files
    - npm run watch

P.S - All templates have own folder /css and /css/sass.
In sass folder we must to add a new our *.+(sass|scss) files and after added we can build or in process working we can watching for files
