1) You need install node.js  (v14.21.3)
2) "npm install" in this directory

If you need to change some files in folder src and to see the changes, you need go to : admin->Configuration->Other config->shop mode : Developer

If you need to build a files: 
    - npm run build
If you need to watch for files and auto building files
    - npm run watch
P.S - This all for admin and site
